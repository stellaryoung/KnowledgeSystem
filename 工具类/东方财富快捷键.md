## 分时图

| 快捷键 | 说明                      |
| ------ | ------------------------- |
| ←与→   | 移动十字光标              |
| ↑与↓   | 查看多日分时              |
| Tab    | 打开/隐藏报价信息         |
| 01或F1 | 个股成交明细              |
| 02或F2 | 个股当日分价              |
| 05或F5 | 切换K线图和分时图         |
| Ctrl+← | 向左快速平移一段K线       |
| Ctrl+→ | 向右快速平移一段K线       |
| Home   | 光标移至当前屏第一根K线   |
| End    | 光标移至当前屏最后一根K线 |
| 08或F8 | K线图下切换分析周期       |
| 92     | 5分钟K线图                |
| 93     | 15分钟K线图               |
| 94     | 30分钟K线图               |
| 95     | 60分钟K线图               |
| 96     | 日线K线图                 |
| 97     | 周线K线图                 |
| 98     | 月线K线图                 |
| 99     | 年线K线图                 |
